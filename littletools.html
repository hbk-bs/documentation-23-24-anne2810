Zeichenfläche

Bei dem Projekt der "little tools" ging es darum eine einfache Zeichenfläche mit unterschiedlichen Werkzeugen zu erstellen. Erstellt wurde die Zeichenfläche durch html, css und js und wurde dann auf eine Webseite geladen. Bei meiner Webseite kann man von einer Farbpalette beliebige Farben auswählen und die Pinselstärke verstellen. Zudem gibt es einen Button der das fertige Bild abspeichert.

Der html Code besteht aus den grundlegenden Elementen, sowie die Links zu der verknüpften css und p5.js Seite. Im "main" wird der Hauptbereich der Seite gezeigt, in dem sich die Zeichenfläche befindet. "sketch" zeigt einen Platzhalter für die Zeichenfläche, die in JavaScript erstellt wird. "aside" setzt eine zusätzliche Nebenleiste welche einen Farbwähler und einen Button, enthält.

der Großteil des codes befindet sich in der sketch.js, welche dafür sorgt, dass auf der Zeichenfläche gemalen werden kann, dabei die Strichstärke und Farbe anpassbar ist und das Bild gespeichert werden kann.

Der Slider wird in dem Fall in die js. platziert und mit einer festen Position auf dem Bildschirm beschrieben, was dazu führt das dieser nicht mehr zu sehen ist, sobald der Bildschirm in einem kleineren Format ist. Die Funktion "mousedragged", sorgt dafür das eine Linien zwischen der aktuellen und der verherigen Position der Maus gezeichnet wird. Der "buttonHandler" sorgt dafür dass das Bild nach dem drücken des speichern Buttons gespeichert wird.

In der style.css wird festgelegt wie die Seite aussehen soll. Es werden die Größen, Positionen und Abstände einzelner Elemente so wie deren Farbe festgelegt. Zudem wird dafür gesorgt, dass der Bildschirm verkleinert werden kann und die Seite trotzdem funktioniert.

Der untenstehende Frame zeigt ein Ausschnitt der Webseite in dem man die Zeichenfläche ausprobieren kann. Da der code zum Teil nicht so geschrieben ist, dass man problemlos den Bildausschnitt verkleinern kann kommt es hier zu einigen Einschränkungen. Einerseits ist der Slider zum verstellen der Pinselgröße nicht sichtbar, da dieser auf einer festen Position liegt, und andererseits sind auf dem Smartphone sämtliche Buttons nur im Querformat benutzbar.
